# JANATPMP (Janat Project Management Platform)

A Gradio-based application for managing codebases.

## Features
- **Codebase Inventory**: Scan and index files across projects (Coming Soon).

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the application:
   ```bash
   python3 app.py
   ```
